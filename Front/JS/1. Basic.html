# Basic

    Using
        use on page :
            <script type="text/javascript">...</script>
        use external :
            <script src="filename.js"></script>

    variable
        for define variables in js we use var keyword
        and with assign value type will set automatically
            var b = "init"; 
            var d = 1;    
            var f = false;
            var h = function(){};
        we can use const keyword for define constants
            const PI = 3.14;
        we can use let keyword for define variable with block scope
            let z = 'zzz';
        values that we assign to variables can be
            false, true                     // boolean
            18, 3.14, 0b10011, 0xF6, NaN    // number
            "flower", 'John'                // string
            undefined, null , Infinity      // special

    Object Data Type
        object is data type like class objects in oop define like below
            var student = {
                firstName:"Jane",           // list of properties and values
                lastName:"Doe",
                age:18,
                height:170,
                fullName : function() {     // object function
                    return this.firstName + " " + this.lastName;
                }
            }; 
        and access object properties like below
            student.age = 19;           // setting value
            student[age]++;             // incrementing
            name = student.fullName();  // call object function

    Array
        define
            var dogs = ["Bulldog", "Beagle", "Labrador"]; 
            var dogs = new Array("Bulldog", "Beagle", "Labrador");
        access and assign
            alert(dogs[1]);
            dogs[0] = "Bull Terier";
        get array length
            dogs.length
        functions on arrays
            >dogs.toString();                        // convert to string: results "Bulldog,Beagle,Labrador"
            dogs.join(" * ");                       // join: "Bulldog * Beagle * Labrador"
            dogs.pop();                             // remove last element
            dogs.push("Chihuahua");                 // add new element to the end
            dogs[dogs.length] = "Chihuahua";        // the same as push
            dogs.shift();                           // remove first element
            dogs.unshift("Chihuahua");              // add new element to the beginning
            delete dogs[0];                         // change element to undefined (not recommended)
            dogs.sort();                            // sort string alphabetically
            dogs.reverse();                         // sort string in descending order

    loops in js
        for (var i of theArray) {}
        theArray.forEach(element => {});
