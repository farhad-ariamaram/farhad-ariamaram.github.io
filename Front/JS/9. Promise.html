# Promise

    Promise is used for async functions in js
    its like call back but is better
    
    for declear a promise
        var promise = new Promise(function(resolve, reject){
            //do something
        });
    and declear in it when should call resolve and when should call reject
        if(things was ok) {
            resolve();
        } else {
            reject();
        }
    and then call promise and use then and catch
        promise.then(function (result) {
            console.log('Promise is successfully.');
        }).
        catch(function (err) {
            console.log('Some error has occured');
        });
