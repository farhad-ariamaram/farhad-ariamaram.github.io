# Extension

    
        every innerphone is an Extension in Issabel

        Innerphone Types
            Softphones : Bria, 3CX, Zoiper, gsWave
            ip-phone (Hardware) => Panasonic(HDV, UT), Cisco, GrandStream, Yealink
        

        Create Extension
            create => PBX -> Extensions -> this options must declear -> User Extension
                                                                        Display Name
                                                                        Secret
    

        Assign Extension
            this options must set on innerphone => Extension number
                                                   Display Name
                                                   Secret
                                                   Issabel server ip
    
                Softphones :                                       
                    in bria UserId should be EXT_NO@ISSABLE_IP
                    in 3CX Extension and ID are extension number and Local IP is Issabel server ip


                ip-phones :
                    first connect it to switch 
                    get phone ip address by phone network setting or ip scanner or mikrotik
                    enter ip-phone settings by entering its ip on web browser

                        Panasonic => 
                                        before going to setting on web we should enable web embedded
                                        Menu -> System Setting -> Network Setting -> Embedded Web -> ON
                                                                or System Setting 

                                        default user is admin and default password is adminpass

                                        VoIP -> SIP setting -> Line

                                        set Issabel server ip on following fields :
                                        Register Server
                                        Proxy Server
                                        presence 
                                        outbound proxy
                                        service domain

                                        Secret : password

                                        PhoneNumber and AuthenticationID : extension number 

                                        Inter Digit Time is timeout between press keys on phone

                        Grand Stream =>
                                        Account -> Acoount1 -> Sip Server : Issabel server ip
                                                            Sip User ID : extension number
                                                            AuthenticationID : extension number
                                                            Secret : password
