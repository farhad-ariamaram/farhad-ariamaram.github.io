# Codec

    
        for compressing and converting voice in phone centers we use codecs

        Codecs Type
            two important codecs that we use are
                G.711 -> also called ulaw an alaw and use over internal communications
                iLBC  -> use in communication between phone center over internet
    
        Setting Codecs in Issabel
            in issabel 2020 -> PBX Configuration -> Asterisk SIP Setting
            in issabel 2017 -> unembeddedIssabelPBX -> Setting -> Asterisk SIP Setting
            in this page we declear what codecs we want use 
        
        Setting Codecs in EXtensions
            in extensions settings in allow and disallow field we declear this extensions
            use what codecs in allow field and don't allow what codecs in disallow field
            if leave empty both field extensions will use all codecs that we select in Asterisk SIP Setting
            Note: allow and disallow also can use in trunk Peer Detail
