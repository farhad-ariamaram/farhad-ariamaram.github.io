# parameters

    for get id in actions we pass its as parameter
        public ActionResult Edit(int id)
    and call this method by one of following urls
        domain/Student/Edit?id=1
        or
        domain/Student/Edit/1
    and if parameters was more than one
        public ActionResult Edit(int id, string name)
    call as follow
        domain/Student/Edit?id=1&name=John

    also we can use parameters in Post actions
        [HttpPost]
        public ActionResult Edit(Student std)
    that we can Incluse and Exclude some properties
        public ActionResult Edit([Bind(Include = "StudentId, StudentName")] Student std)
        public ActionResult Edit([Bind(Exclude = "Age")] Student std)
    and inputs must have exact same name of Student model properties
    or we can use FormCollection for access by input name 
        [HttpPost]
        public ActionResult Edit(FormCollection values)
        {
            var name = values["StudentId"];
        }
