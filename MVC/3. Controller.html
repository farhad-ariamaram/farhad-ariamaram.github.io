# Controller

    Controllers define in Controllers Folder
    every class in this folder role as a controller that extend Controller base class
    and its name should end with Controller like HomeController

    every controller have methods call Action
    that in routing we call them 
    and if we dont specify action in url by default Index will call (if in routing specify default action be Index value)

    Actions return type is ActionResult that can be following types
        ViewResult              -> HTML                         -> View()
        ContentResult           -> String                       -> Content()
        FileContentResult       -> File                         -> File()
        JavaScriptResult        -> javascript script            -> JavaScript()
        JsonResult              -> json use by ajax or APIs     -> Json()
        RedirectResult          -> Redirect to a new URL        -> Redirect()
        RedirectToAction        -> redirect to other controller -> RedirectToAction()
        PartialViewResult       -> Partial view                 -> PartialView()
        EmptyResult             -> No Response
        HttpUnauthorizedResult  -> http 403 code

    normally we return a view in Actions
    if we dont specify view name in View() by default go to View that has same name by action
    or we can specify what view we want to return
        View("ViewName") -> ViewName should be in same action controller folder name in view
        for example
            HomeController -> Index -> return View("TestView")
            View -> Home -> TestView.chtml
        if we want to return another controller action View we use RedirectToAction
            return RedirectToAction("ViewName", "ControllerName");

    with [ActionName("Find")] attribute we can specify other name for action

    with [NonAction] attribute we can declear a normal method that is not action

    with [HttpPost], [HttpGet], [HttpPut] and ... we can specify ActionVerb of Action 

    for pass model to a view we pass it by View(Model)
        public ActionResult Index()
        {
            //fetch students from the DB using Entity Framework here
            return View(studentList);
        }
    and in the view receive it by @model
    in first line of view we specify model type
        @model IEnumerable
    use model in foreach
        @foreach (var item in Model)
    use as lamda
        (model => model.StudentName)
    use by @Model
        @Model.property.toString();
