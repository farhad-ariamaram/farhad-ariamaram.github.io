# Routing

    Routes config in App_Start -> RouteConfig.cs class
        
        routes.MapRoute(
            name: "Default",
            url: "{controller}/{action}/{id}",
            defaults: new { controller = "Home", action = "Index", id = UrlParameter.Optional }
        );
        
    its mean urls should be following format
        http://doman/controllerName/actionName/id
    and if any of sections dont enter by default will be following values
        controller = "Home"
        action = "Index"
    and id is optional
        id = UrlParameter.Optional

    if a new route add to this class file upper route has priority

    in Global.asax we should register this route class by following code (by default has done)
        RouteConfig.RegisterRoutes(RouteTable.Routes);
