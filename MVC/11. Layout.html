# Layout

    we define a layout in shared folder of view and use it in all views for repeted area
    in layout RenderBody() section will replace by view html codes
    and we can declear RenderSection(string name,bool required) to use them in views

    also we can change a view layout by following code in view
        @{
            Layout = "~/Views/Shared/_myLayoutPage.cshtml";
        }
