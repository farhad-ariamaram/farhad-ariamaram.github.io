# ViewBag

    we can use ViewBags as temporary variables between view and controller

        in controller :
            public ActionResult Index()
            {
                ViewBag.TotalStudents = studentList.Count();
                return View();
            }

        in view:
            <label>Total Students:</label>  @ViewBag.TotalStudents

    but in refresh or redirections they will gone 

    ViewData is same as ViewBag with different code
        in controller :   
            ViewData["students"] = studentList;
            ViewData.Add("Id", 1);

        in view :  
            @std.StudentName

    TempData is for storing data between controller action methods or views 
    but it save as Sessions and be in memory longer
        TempData["name"] = "Bill";
        @TempData["name"]
